¿CUÁL ES EL PROBLEMA DE LA CONVERSIÓN IMPLÍCITA?

		var num = prompt("Ingresa un Numero");
		alert(num + 2);

¿Si el usuario ingresa 2, cuál será la respuesta?

CÓMO LO RESOLVEMOS?

Utilizando parser, en este caso parseInt(“string”)

		var num = prompt("Ingresa un Numero");
		num = parseInt(num);
		alert(num + 2);

Podemos hacer lo mismo compactando el codigo

		var num = parseInt(prompt("Ingresa un Numero"));
		alert(num + 2);

CONDICIONES: ELSE IF

CONDICIONES: OPERADOR TERNARIO

El operador ternario es otra forma de escribir una condición if else pero con menos código, la lógica es la siguiente: 

		condicion ? Verdadero : Falso

		var edad=19;
		var resultado = edad >=18 ? "Eres mayor de Edad" : "No Eres mayor de Edad":
		alert(resultado);

FOR

Crea un bucle que consta de tres expresiones opcionales,encerrados entre paréntesis y separados por punto y coma,seguido de una sentencia ejecutada en el bucle.

Sintaxsis:

		for(Inicializacion,Condicion,Incremento){
			Declareaciones
		}

Ejemplo

		var numero = prompt("Ingresa un numero para Factorial");

		var factorial =1;
		for (var i = 1; i <= numero; i++) {
		 	factorial=factorial*i;
		}

	 	alert("El factorial de "+ numero+" es: "+factorial);

Funciones

Cuando se desarrolla una aplicación compleja, es muy habitual utilizar una y otra vez las mismas instrucciones. Un script para una tienda de comercio electrónico por ejemplo, tiene que calcular el precio total de los productos varias veces, para añadir los impuestos y los gastos de envío.

Las funciones se recomiendan usar cuando se vaya a reutilizar el codigo.

Veamos el siguiente codigo.

		var resultado=0;
		var numero1 = 3;
		var numero2 = 5;
				
		//	Se	suman	los	números	y	se	muestra	el	resultado
		resultado = numero1 + numero2;
		alert("El	resultado	es	" + resultado);
				
		numero1 = 10;
		numero2 = 7;
				
		//	Se	suman	los	números	y	se	muestra	el	resultado	
		resultado = numero1 + numero2;
		alert("El	resultado	es	" + resultado);
				
		numero1 = 5;
		numero2 = 8;
				
		//	Se	suman	los	números	y	se	muestra	el	resultado
		resultado = numero1 + numero2;
		alert("El	resultado	es	" + resultado);

Ahora lo hagamos con funciones.

	
		function fntestSuma(numero1,numero2) {
			var resultado=0;
			
			resultado = numero1 + numero2;
			alert("El resultado es " + resultado);
		}

		var numero1 = 3;
		var numero2 = 5;
				
		//	Se	suman	los	números	y	se	muestra	el	resultado
		fntestSuma(numero1,numero2);
				
		numero1 = 10;
		numero2 = 7;
				
		//	Se	suman	los	números	y	se	muestra	el	resultado	
		fntestSuma(numero1,numero2);
				
		numero1 = 5;
		numero2 = 8;
				
		//	Se	suman	los	números	y	se	muestra	el	resultado
		fntestSuma(numero1,numero2);
